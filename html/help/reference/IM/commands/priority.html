<&| /help/_elements/wrapper, title => 'IM Command: Priority' &> %# terse: priority TASKS NEWPRIORITY. See or change priority of tasks. http://tinyurl.com/yrx4ah

Lets you see or change the priority of one or more tasks.

If you don't provide a new priority it will show you the priority of the tasks you give.

Priorities can be numeric (1-5 with 5 being highest), or: lowest, low, normal, high, highest. You may also use the braindump-like priorities: --, -, +, ++, !, or !!.

See also help context if you're wondering how the last example works!

Examples:

  • priority #9AN highest
  • priority ZEY !!
  • priority #CLR
  • priority these 3
