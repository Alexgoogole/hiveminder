The command to create new tasks. Works identically to the braindump box on every Hiveminder page.

For rapid-fire task creation you can type in just create with no task and you'll enter modal create mode. Every subsequent message is collected and made into one big braindump at the end. You finish the create by sending done on its own line. You can also cancel the create with cancel on its own line.

You can also use create [tags] [due: tomorrow] and so on to apply the tags and other attributes to all tasks created during the modal create.

Examples:

  • create make dinner tonight [grocery]
  • c eat dinner tonight [food]
  • modal create:
    • create [food dinner] [due tonight]
    • buy dinner [grocery money]
    • eat dinner
    • do dishes [chore]
    • done
