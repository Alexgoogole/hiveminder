To add Hiveminder's IMAP server to your Thunderbird client:

  1. Select Account Settings... from the Edit menu.
  2. Click Add Account... near the bottom left of the window.
  3. Select Email account when prompted for they type of account.
  4. Enter your name, and the email address you used to register for Hiveminder.
  5. Select IMAP as the type of server, and enter hiveminder.com as the incoming server name.
  6. Input your Hiveminder email address as your incoming user name.
  7. Enter a name for this account, like Hiveminder!
  8. Once you've checked the details, you're nearly done.
  9. Select Server Settings in the left pane, and change the Use secure connection selection on the right to SSL.
  10. You're all set to go with Hiveminder IMAP!
