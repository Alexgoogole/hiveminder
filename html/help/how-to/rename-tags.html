Tags are really useful for finding tasks. However, sometimes you want to change the name of a tag. Hiveminder supports this through its <% Jifty->web->link(url => "/help/reference/tasklists/bulk-update.html", label => "bulk update") %> feature.

Say you want to rename the baby tag to adult.

  1. Go to your <% Jifty->web->link(url => "/todo", label => "todo list") %>.
  2. Click the tag you want to rename, baby, in the tag cloud on the left. This will show you all tasks with that tag.
  3. Click bulk update at the top of your tasklist. This will open a new pane.
  4. If you have more than one page of tasks, click all in the paging links in the top right to show all tasks instead of just the first page. This will make bulk update update all tasks.
  5. Put your new tag, adult, into the "Add Tags" text box.
  6. Put the old tag, baby, into the "Remove Tags" text box.
  7. Click the Save Changes button to perform the bulk update.
