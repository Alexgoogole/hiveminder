To add Hiveminder's IMAP server to your Outlook Express client:

  1. Select Accounts... from the Tools menu.
  2. Select Mail from the Add button on the right.
  3. Enter your name as the Display name.
  4. Enter your email address as the Email address.
  5. Select IMAP as the type of server, and enter hiveminder.com as the incoming mail server. Hiveminder does not provide an outgoing mail server; you should enter your regular email account's outgoing mail server here, not mail.example.com as shown here. If you don't know what your outgoing mail server is, ask your local system administrator (nicely!) for what it should be.
  6. Enter the email address you used to register for Hiveminder as the Account name, and your Hiveminder password as the Password.
  7. You're nearly done -- click Finish.
  8. Outlook will prompt you if you wish to view a list of available folders for the account; click No.
  9. Click on the hiveminder.com account, then click Properties.
  10. Click the Advanced tab, and check the This server requires a secure connection (SSL) checkbox.
  11. Outlook will prompt you if you would like to refresh your folder list. Click Yes.
  12. You're all set to go with Hiveminder IMAP!
