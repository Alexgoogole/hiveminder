Gives you a random task. Very handy if you have a lot of little things that need doing, but you're not sure where to start. This gives you only tasks that would show up in your todo list (meaning incomplete tasks that aren't hidden, and so on).

You can also use random search terms to narrow down the pickins.

Examples:

  • random
  • random book
