Hiveminder's IM system supports the notion of context. Isn't it annoying to have to tell me what tasks you're talking about all the time?

create finish reading Dune [book dune]
due #6NCO by thursday
comment #6NCO wow what a great book!
tag #6NCO [favorite]
done #6NCO

Instead, with the context system, you can use this or these (makes no difference which one you use) in place of task IDs, like so:

create finish reading Dune [book dune]
due this by thursday
comment this wow what a great book!
tag this [favorite]
done this

The show command will list the tasks are in your current context.
