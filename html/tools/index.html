There are lots of ways you can work with Hiveminder!

Hiveminder in your browser

If you use Firefox or IE 7, you can Braindump or Search Hiveminder from your browser's search box.

Add Braindump to your browser.

Add Hiveminder Search to your browser.

todo.pl
Command Line Interface

If you're a fan of Todo.sh, don't miss our todo.pl. You can import an existing Todo.sh text file. If you have problems with usage, read this instruction first. This program is MIT Licensed.

To use todo.pl, please install App::Todo from CPAN. From the command line, you should be able to run cpan -i App::Todo.

iMinder, an iPhone app

Kris Arnold has written iMinder, a free app to allow you to access and sync your tasks with your iPhone. You can download it from the App Store.

Hiveminder Braindump, an Android app

Marc Dougherty created Hiveminder Braindump, a simple Android app to let to jot down your tasks. You can search, view, and create tasks, but editing isn't supported at this time. Get it from the Marketplace!

Hiveminder Cloud Tasks, a Palm webOS app
Paul Miller created this app enabling you to access your Hiveminder tasks from your Palm webOS device. It's currently in beta and only allows viewing tasks, but he promises that there's much more to come. Check it out on the app catalog!
Quicksilver

Zak Greant has a quick tip for using Hiveminder via IM from Quicksilver.

Kris Arnold also built a Quicksilver integration using todo.pl.

Hiveminder API

Perhaps none of our existing tools meet your needs exactly. You could use our API to write your perfect interface to Hiveminder. Please join our mailing list to ask questions or to share in your creations!

If you speak Perl, we have a Net::Hiveminder module that makes using our API even easier.

Sync with Outlook

Nathan Vonnahme has written a program to sync your Hiveminder tasks with Outlook, which in turns lets you sync from Outlook to your Palm device.

Sync with RT, Google Code, Trac, and more
Peer-to-peer bug tracking and syncing

We've also created SD (Simple Defects) which leverages Prophet to provide a simple bug tracker which can sync with other SD databases and a variety of services. SD makes bidirectional sync between RT and Hiveminder possible.
