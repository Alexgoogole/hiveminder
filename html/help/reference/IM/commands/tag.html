<&| /help/_elements/wrapper, title => 'IM Command: Tag' &> %# terse: tag TASKS [TAGS]. Add tags to the given tasks. http://tinyurl.com/2ed98d

Lets you add tags to one or more tasks.

See help untag if you would like to remove tags.

See help tags if you want to see the list of the tags you've already used.

See also help context if you're wondering how the last example works!

Examples:

  • tag #6NCD with bowie knife
  • tag 6NCE 6NCF [music bowie]
  • tag this with context
