<&| /help/_elements/wrapper, title => 'IM Command: Note' &> %# terse: note TASK NOTE. Add a note to a task. http://tinyurl.com/2dqmaz

Lets you add (or see) notes on a task. Notes are just an extension of the task summary. For discussion of the task, you want comments.

To add a note, use note TASKS: NEW NOTE.

To clear notes, use note clear TASKS.

To read a note, use note read TASK.

See also help context if you're wondering how the last example works!

Examples:

  • note #DINNER: I have chicken wings and leftover spaghetti
  • note read #DINNER
  • note clear #DINNER
  • note #HELP: I'm using version 7.1
  • note this: I'm using version 7.1
  • note: I'm using version 7.1
