Bulk Update allows you to update several tasks at the same time. Use the "Search" function to narrow down the list down to the tasks you want to update, and then use the "Not this" links to remove tasks you weren't able to sort out of your list using "Search." You can now change the due date, the tags, the priority or any other task detail on all of those tasks all at once.

You can use Bulk Update to, for example, change the tag "money" to "spending":

  1. Go to your <% Jifty->web->link(url => '/todo', label => 'todo list') %>.
  2. Click the money tag in your tag cloud.
  3. Click Bulk Update at the top of the task list.
  4. Click All at the top right to show all tasks instead of just the current page.
  5. Put "spending" into Add Tags in the Bulk Update window.
  6. Put "money" into Remove Tags in the Bulk Update window.
  7. Click Save Changes.
