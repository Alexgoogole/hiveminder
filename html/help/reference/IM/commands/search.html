<&| /help/_elements/wrapper, title => 'IM Command: Search' &> %# terse: search QUERY. Lets you search *all* of your tasks, not just the ones on your todo list. http://tinyurl.com/yucjr2

Basically the same as the todo command but less restrictive. Todo won't show you finished tasks, or tasks that people have assigned to you but that you haven't accepted.

The search command only works for task summaries, tags, etc, similarly to the search box in Hiveminder's web interface. If you want to search for more complicated criteria such as priority or due date, see help filter.

Examples:

  • search birthday
  • find archery
  • /weekend
