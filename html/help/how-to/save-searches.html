To make our powerful search feature even more useful, pro users can now save their searches. Spend less time searching, more time doing!

You can create a saved search in three easy steps.

  1. Do your search. How about "All tasks above normal priority with tag @work"?
  2. Click the Save List link above the list of tasks.
  3. Type in a name for your search. (such as "Urgent, for work")

Using saved searches is easy too.

  1. Click the arrow next to To Do in the top menu bar.
  2. Click the saved search you want to see.
